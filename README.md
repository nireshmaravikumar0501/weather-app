# Advanced Weather App

A responsive weather application built using HTML, CSS and JavaScript that allows users to search for a city and view live weather information using the Open-Meteo API.

## Features
- Search weather by city
- View live temperature
- Check wind speed
- Animated weather icons
- Modern glassmorphism UI
- Responsive design

## Tech Stack
- HTML
- CSS
- JavaScript
- Open-Meteo API

## What I Learned
- DOM Manipulation
- Fetch API
- Working with JSON data
- API integration
- Event handling
- Responsive UI design

## Project Structure
weather-app/
├── index.html
├── style.css
├── script.js

## How to Run
1. Download or clone this repository
2. Open index.html in browser
3. Enter a city name
4. View weather details

## Future Improvements
- 5-day weather forecast
- Dynamic weather backgrounds
- More animated weather icons
- Search history
